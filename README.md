# RusGator — Викторина «Русский язык и литература»

Интерактивная викторина для открытого урока. Сетка 2×3 — каждая ячейка скрывает часть фразы, которая открывается при правильном ответе.

## 🚀 Запуск

Откройте `index.html` в браузере — никакой сборки не требуется.

## 🌐 Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Залейте все файлы:
   ```
   git init
   git add .
   git commit -m "init: RusGator Quiz v1.1"
   git remote add origin https://github.com/ВАШ_ЛОГИН/RusGator.git
   git push -u origin main
   ```
3. В настройках репозитория → **Settings → Pages → Source: main / root**
4. Через ~1 минуту сайт будет доступен по адресу `https://ВАШ_ЛОГИН.github.io/RusGator/`
